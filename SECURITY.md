# Security Policy

## Supported Versions

**The WallaBMC project is in early stages, and the release and security
support process has not been established yet. We take security seriously and
aim to fix, but there are no supported versions as such. Fixes will be made to
the head of the tree.**

**Do not deploy WallaBMC on the public internet or allow access to it from untrusted
users.**

## Reporting a Vulnerability

Since this is still pre-alpha software, if you discover a security vulnerability,
please report this as a public github issue.
